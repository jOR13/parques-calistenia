# 🏋️‍♂️ Parques de Calistenia - Ciudad Juárez

Una aplicación web hermosa y sencilla para encontrar el parque de calistenia más cercano a tu ubicación en Ciudad Juárez, Chihuahua.

## ✨ Características

- 📍 **Geolocalización automática**: Encuentra tu ubicación y muestra los parques más cercanos
- 🎯 **Cálculo de distancias**: Ordenados por proximidad a tu ubicación
- 📱 **Diseño responsivo**: Se ve perfecto en móviles, tablets y desktop
- 🗺️ **Integración con Google Maps**: Obtén direcciones con un click
- 💬 **WhatsApp integration**: Contacta fácilmente para más información
- 🎨 **UI moderna**: Diseño atractivo con gradientes y animaciones

## 🚀 Instalación y uso

```bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Construir para producción
npm run build
```

## 📋 Parques incluidos

La aplicación incluye **19 parques de calistenia** en Ciudad Juárez con:
- Ubicaciones exactas con coordenadas GPS
- Horarios de clases
- Días de la semana disponibles
- Información de instructores
- Direcciones completas

## 🛠️ Tecnologías utilizadas

- **React 18** - Biblioteca de interfaz de usuario
- **Vite** - Herramienta de construcción rápida
- **Tailwind CSS** - Framework CSS utilitario
- **Geolocation API** - Para obtener la ubicación del usuario
- **Google Maps** - Para direcciones y navegación

## 💪 Clases gratuitas

Todas las clases de calistenia son **completamente gratuitas** y están abiertas para toda la comunidad. Los horarios y días varían según el parque y el instructor.
